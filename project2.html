Emergency Relocation Model

Backstory

This project was part of a datathon that tasked me and other teammates with creating a model to estimate areas in Texas, Georgia, and Florida that are low income and vulnerable to flooding.

I've also included various images mapping these vulnerable areas, along with viable areas to relocate these vulnerable communities

This project demonstrates the use of various datasets and machine learning techniques to address this real-world problem. The insights gained from this analysis can help in making informed decisions to support vulnerable communities in these areas.

For more details, visit the project's GitHub repository: Datathon 2024.
