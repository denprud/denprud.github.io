Box Words

Backstory

As part of a personal project for the Buildspace cohort, I've built a full-stack web-based word game that allows the user to fill in randomly generated puzzles.

The puzzle can be solved by filling the box with words that also adhere to the rules that correspond to each side of the box.

I've also engineered a MongoDB database to capture user responses and analyze puzzle statistics.

Here is a link to try the game out! Box Words.
